Identity
=======
- A react.js project which handle a multi-page form to register the info of a person

[Live Preview](https://ubaidurrehman1.github.io/identity/)
----------------------------------------------------------

How to Run
----------
- `git clone https://github.com/UbaidurRehman1/identity.git`
- `cd identity`
- `npm install`
- `npm start`
